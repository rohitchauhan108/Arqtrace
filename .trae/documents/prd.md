## 1. Product Overview
The project is a high-fidelity clone of the "Floens – Tiling & Flooring Service" website layout. It aims to provide a professional, modern, and responsive interface for a flooring service business, matching the exact visual style and structure of the reference site.
- Target Users: Flooring service providers, interior designers, and homeowners looking for renovation services.
- Value Proposition: A clean, industry-specific design that establishes trust and showcases services effectively.

## 2. Core Features

### 2.1 Feature Module
1. **Home Page**: The main landing page comprising all key sections of the business.
2. **Navigation**: A sticky header with multi-level menus and contact info.
3. **Hero Slider**: An interactive slider showcasing high-quality interior flooring shots.
4. **Service Showcase**: A grid of services offered with detailed descriptions and icons.
5. **Project Portfolio**: A visual grid of completed projects.
6. **Information Sections**: About Us, Stats Counter, and Testimonials.
7. **Blog/News**: Latest updates and articles.
8. **Footer**: Comprehensive site information and quick links.

### 2.2 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| Home Page | Hero Slider | Auto-playing slider with animated text and primary CTAs. |
| Home Page | About Section | Asymmetrical image layout with detailed service features. |
| Home Page | Services Grid | Responsive cards for different flooring types (Tiling, Wood, etc.). |
| Home Page | Stats Section | Animated counters for client satisfaction and projects. |
| Home Page | Project Grid | Hover-effect portfolio showing project names. |
| Home Page | Testimonials | Centered slider for client feedback. |
| Home Page | Footer | Multi-column layout with branding and contact details. |

## 4. User Interface Design

### 4.1 Design Style
- **Primary Color**: Terracotta/Burnt Orange (`#BC8B57`) for accents and buttons.
- **Secondary Colors**: Dark Charcoal (`#1F1F1F`) for headings, Soft Grey (`#707070`) for body text.
- **Backgrounds**: Off-white/Cream (`#F8F7F2`) and Pure White.
- **Typography**: Plus Jakarta Sans (Bold for headings, Regular for body).
- **Layout**: Modern boxed/wide layout with generous whitespace and grid-based sections.

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| Home Page | Hero Section | Large images, staggered text animations, prominent buttons. |
| Home Page | Services | Icon-based cards with hover lift effects. |
| Home Page | Stats | Parallax background or solid color with clean typography. |

### 4.3 Responsiveness
- Desktop-first design with mobile-adaptive layouts.
- Hamburger menu for mobile navigation.
- Grid columns stack on smaller screens (3 -> 2 -> 1).
